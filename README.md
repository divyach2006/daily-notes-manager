# 🌸📝 Daily Notes Manager

A beautiful and user-friendly Notes Management Application built using **HTML, CSS, JavaScript, Node.js, and Express.js**. This project helps users save, search, view, and delete daily notes with ease.

---

## ✨ Features

🌷 Save notes with date

🔍 Search notes by date

📖 View saved notes instantly

🗑️ Delete notes easily

💖 Attractive and responsive pink-themed UI

⚡ Fast and simple note management

---

## 🛠️ Tech Stack

🎨 HTML5

🎀 CSS3

⚙️ JavaScript

🟢 Node.js

🚀 Express.js

📂 File System (fs Module)

## 🚀 Getting Started

### 1️⃣ Install Dependencies

```bash
npm install
```

### 2️⃣ Run the Application

```bash
node app.js
```

### 3️⃣ Open in Browser

```text
http://localhost:3000
```
